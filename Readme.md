# ICE-Tracker (Alpha)

A lightweight Python GUI application designed to track, log, and visualize Synthego ICE (Inference of CRISPR Edits) Knockout Scores over time.

> **Status:** 🚧 **Alpha Phase** (v0.1.0) — Main GUI (`ICEVisualizerapp`) initialized. Active development in progress.

---

## 📌 Overview

This tool simplifies data entry and time-series visualization for CRISPR knockout experiments analyzed via Synthego ICE. It allows researchers to intuitively input ICE scores across specific days and automatically generates progression curves for longitudinal analysis.

---

## ✨ Features

### Current (v0.1.0 - Alpha)
- [x] Graphical User Interface (`ICEVisualizerapp`) launch window.

### Planned / Roadmap 🛠️
- [ ] **Data Entry:** Input fields for Synthego ICE Knockout Scores and time intervals (Days).
- [ ] **Visualization:** Automated plotting of score progression curves over time.
